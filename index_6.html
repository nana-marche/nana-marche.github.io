TYPO SITE
画像より、
フォントと余白で勝負する。
SMARTPHONE FIRST / TYPE ONLY

余白とフォントだけで、サイトはどこまで成立するか。

写真も、アイコンも、装飾も、いったん脇に置いて。 タイポグラフィと余白だけを使って、どこまで世界観を出せるかを試した スマホファーストのサンプルです。

スクロールのたびに「一画面一メッセージ」。読み終えたあとに、少しだけ 記憶に残るような言葉の配置を目指しています。

CONCEPT

「何も置かない」から
見えてくるものがある。

画像や装飾をいったん手放してみると、残るのは「言葉」と「余白」です。 この2つだけでページを組むと、何を大きくし、何を小さくし、 どこで息継ぎをさせるか――そんな基本設計がクリアになります。

読みやすさと「間」のデザイン

文字サイズを少しだけ大きく、行間を少しだけ広く。 そのうえで、段落のあいだにきちんと「間」を置くことで、 情報は詰め込まずに、静かに届くようになります。

あえて説明しすぎない文章量

すべてを書き切るのではなく、読み手に1〜2歩だけ想像させる余白を残す。 その感覚は、レイアウトの余白感ともよく似ています。

このページで試していること

・見出しは明朝体でエッジを効かせる
・本文は角ゴでフラットに整える
・欧文は別フォントで“音の違い”をつくる
・上下左右の余白を「多いかな？」くらいまで引き伸ばす

VISION

TEXT FIRST, THEN EVERYTHING.

いい言葉は、
いいデザインと同じくらい、
人の記憶に残る。

だからこのサイトでは、伝えたいことをできるだけ すっきりと言葉で描ききることから始めています。 色や装飾は、そのあとについてくるものとして。

「何を話すか」が決まれば、
「どう見せるか」は、あとからいくらでも整えられる。
その逆は、なかなかうまくいきません。

DESIGN NOTES

テキストだけのサイトを
組むときに意識した3つのこと。

実際の案件に転用しやすいように、「すぐに使える視点」だけを 3つに絞りました。どれも、画像があってもそのまま活かせる考え方です。

  • POINT 01
    一画面で伝えるメッセージはひとつだけ

    ヒーローでは「コンセプト」、次は「考え方」、その次は「具体的なポイント」―― というように、スクロールごとに役割を分けています。

  • POINT 02
    フォント同士の「キャラの違い」をはっきりさせる

    見出しと本文、欧文と和文で、それぞれフォントの役割を分けています。 同じ情報でも、フォントのキャラクターが変わるだけで印象は大きく変わります。

  • POINT 03
    余白もデザインの一部として数える

    テキストの量を増やす前に、「ここにもう一行の余白を足せるか」を 一度だけ考えてみる。余白を削るよりも、足す方向に振ると、 仕上がりが一段落ち着きます。

FLOW

つくりかたも、
シンプルであっていい。

実際にページをつくるときのステップも、できるだけ少なく。 「話す → まとめる → 組んでみる」という、シンプルな流れで 形にしていきます。

  1. 01
    HEAR
    ざっくりと、話を聞かせてもらう

    まずはオンラインやチャットで、「誰に」「何を」「どう感じてほしいか」 をラフに共有してもらうところから始めます。資料がなくても大丈夫です。

  2. 02
    EDIT
    テキストと構成を、一緒に整理する

    いただいた言葉をベースに、見出し・本文・補足などの役割を分けていきます。 「削る」「言い換える」も、この段階で遠慮なく提案します。

  3. 03
    LAYOUT
    レイアウトに落とし込み、微調整する

    実際の画面に文字を流し込み、行間・余白・強弱を整えます。 スマホでの読み心地を優先しつつ、PCでも美しく見えるように仕上げます。

  4. 04
    TUNE
    公開後も、少しずつチューニング

    実際に使ってみて、「もっとこうしたい」が出てきたら、 言葉や構成を少しずつアップデートしていきます。つくって終わりにはしません。

LAYOUT STUDY

テキストだけで、
どこまでレイアウト遊びができるか。

罫線やボックスを使いすぎず、文字サイズと配置だけで 「分節」と「強弱」をつくる小さなスタディです。 実案件のセクション構成にも、そのまま転用できます。

SCENE 01

読み始める前に、
空気感だけ先に伝えておく。

セクションの冒頭で、細かい説明よりも「どんなトーンの話か」を 先に示しておくと、そのあとの文章が読みやすくなります。

SOFT INTRO

まずは、
空気をわかってもらう。

SCENE 02

あえて情報量を
一段だけ落としてみる。

すべてを詰め込むのではなく、「ここではこれだけ伝える」と 決めることで、印象に残る断面が生まれます。

LESS BUT CLEAR

情報を減らすと、
輪郭がはっきりする。

SCENE 03

「まとめ」ではなく、
次への余白を残して終わる。

完璧に締めくくるのではなく、「続きは別のページで」と 余白を残すことで、体験全体の設計につながります。

NEXT HOOK

ここで終わらせずに、
つづきを用意しておく。

NEXT STEP

あなたの文章を、
そのままレイアウトしてみる。

実際にこのテイストでページを作ってみたいときは、 まずはラフなテキストだけを送ってもらえれば十分です。 見出しの分割や、余白の置き方も含めて一緒に組み立てていきましょう。

用意してもらえると嬉しいもの

・だいたいの構成案（セクションの順番など）
・必ず伝えたい一文（キャッチコピーのタネ）
・ページを見た人に「どう感じてほしいか」の一言メモ
